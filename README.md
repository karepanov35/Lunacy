# Lunacy
![Lunacy Logo](./assets/446C9006-6976-4F8A-B47F-D2AFD72F3F61.png)


  ________________
  
🇺🇸EN: Lunacy — server software for Minecraft Bedrock Edition, built on top of PocketMine-NetherGames.

Our modification runs on PHP language, version 8.1+, and also supports all new versions of Minecraft Bedrock from 1.20.0 up to 1.26.

🇷🇺RU: Lunacy — Серверный софт, для Minecraft Bedrock Edition, создано на основе PocketMine-NetherGames.

Наша модификация работает на PHP языке на версии от «8.1+», так же поддерживает все новые версии Minecraft Bedorck начиная с 1.20.0 до 1.26.

❗️We didn't list all the changes; you can see them in the releases.
